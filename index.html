Stock-Quote

一个简洁实用的命令行 + 图形界面行情查询工具（股票・外汇・加密货币）

核心功能 快速上手 使用示例 查看源码

核心功能

多市场支持

支持查询 A 股、港股、美股、外汇 (如 USDJPY)、主流加密货币 (BTC、ETH ↓) 价格。

实时刷新

可自定义刷新间隔，动态更新行情，适合即时跟踪。

双界面支持

提供命令行界面（CLI）和图形界面（GUI），满足不同使用场景的需求。

自选管理

自动生成并管理自选列表 (favorites.json)，便于长期追踪关注的股票或货币。

跨平台兼容

在 Windows、macOS、Linux 上均可运行，使用 Python 开发。

可打包执行

支持使用 PyInstaller 打包为独立可执行文件，方便分享与部署。

快速上手

  1. 克隆或下载仓库到本地。
  2. 确保已安装 Python 3 环境。
  3. 执行安装依赖：
pip install -r requirements.txt

使用示例

命令行模式（CLI）

python stock_quote.py SH513100 BTC USDJPY
# 设置刷新间隔为 10 秒：
python stock_quote.py -i 10 SH513100 BTC

图形界面（GUI）

python stock_quote_gui.py

启动 GUI，即可通过可视界面管理自选、设置刷新、手动刷新等。
